Maven HTMLCompressor Plugin

  • Last Published: 2023-04-22|
  • Version: 1.9.1
  • Project Documentation
  • Project Information
  • Project Reports

htmlcompressor:html

Full name:

com.github.hazendaz.maven:htmlcompressor-maven-plugin:1.9.1:html

Description:

Compress HTML files.

Attributes:

  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: compile.
