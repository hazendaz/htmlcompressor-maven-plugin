Maven HTMLCompressor Plugin

  • Last Published: 2023-04-22|
  • Version: 1.9.1.1
  • Project Documentation
  • Project Information
  • Project Reports

htmlcompressor:xml

Full name:

com.github.hazendaz.maven:htmlcompressor-maven-plugin:1.9.1.1:xml

Description:

Compress XML files.

Attributes:

  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: compile.

Optional Parameters

Name Type Since Description
<enabled> boolean - if false all compression is off (default is true).
Default value is: true.
User property is: htmlcompressor.enabled.
<encoding> String - Charset encoding for files to read and create.
Default value is: UTF-8.
User property is: htmlcompressor.encoding.
<fileExt> String[] - file types to be processed.
User property is: htmlcompressor.fileExt.
<removeComments> boolean - if false keeps XML comments (default is true).
Default value is: true.
User property is: htmlcompressor.removeComments.
<removeIntertagSpaces> boolean - removes iter-tag whitespace characters (default is true).
Default value is: true.
User property is: htmlcompressor.removeIntertagSpaces.
<skip> boolean - Skip run of plugin.
Default value is: false.
User property is: skip.
Alias is: skip.
<srcFolder> String - source folder where xml files are located.
Default value is: ${project.basedir}/src/main/resources.
User property is: htmlcompressor.srcFolder.
<targetFolder> String - target folder where compressed xml files will be placed.
Default value is: ${project.build.directory}/classes.
User property is: htmlcompressor.targetFolder.
